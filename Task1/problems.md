 # Проблемные места

  ## Нужную отчётность сложно построить за разумное время
  ## Очень много данных (сотни терабайт)
  ## Много вариантов использования данных
  ## Большое количество трансформаций
  ## Неудовлетворительный time-to-market
  ## Снижению производительности
  ## Сложные отчёты можно ждать часами
  ## MS SQL 2008 (EOL 2019)
  ## Все данные лежат в одной монолитной DWH
    Персональные данные
        Данные по клиентам.
        Данные по персоналу больницы.
    Медицинские данные
        Медицинские карты и истории болезни, в том числе — данные исследований, выполненных в ходе лечения.
    Финансовые данные
        Финансовая история.
        Счета.
        Данные о кредитах.
        Данные по инвентаризации.
        Финансовая отчётность 
    Иные данные 
        много другой информации.

  Для приоретизации

  - Запланированы конкретные проекты в доменах по развитию «витрины данных».

  Целевая структура
  - портал самообслуживания
